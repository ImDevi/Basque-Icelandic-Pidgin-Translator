**Basque-Icelandic-Pidgin-Translator**
-----------------------------------------------------------------------------------------------
The **Basque-Icelandic Pidgin Translator** is an automated translation tool for translating into
the 17th century language of Basque whalers - Basque-Icelandic Pidgin.
